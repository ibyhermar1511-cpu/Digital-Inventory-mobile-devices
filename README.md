# 📦Digital Inventory Control System through QR Scanning and Mobile Devices

## Description
This project consists of the development of a digital system for inventory management using QR scanning technology and mobile devices. The objective is to optimize the control of incoming and outgoing items and the registration of products, reducing human errors and improving efficiency in warehouse processes.

##🎯Objective 
### General Objective
Develop a digital system that allows the efficient management of inventories through the use of QR codes and mobile devices.

### Specific objectives
- Register products using QR codes
- Facilitate the control of entries and exits
- Check inventory in real time
- Reduce errors in manual data handling
- Improve the organization of the warehouse

##🛠️Technologies to use
- Python 🐍
- QR code readers
- Mobile devices
- Database (SQLite)
- Git and GitHub

##⚙️Main functionalities
-📥 Materials registration
- 📥Control of outputs
- 🔍Inventory inquiry
- 📱QR Code scanning
- 📊 Real-time updating
